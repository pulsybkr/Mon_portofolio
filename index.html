Ce que je sais faire

Le wireframe

Pour un site ergonomique, il faut réfléchir à la structure des différentes pages. Le wireframe permet de définir le squelette des interfaces du site. C’est une sorte de schéma qui montre l’agencement des blocs et des contenus. Le résultat de cette étape servira de base pour la création des maquettes.

Maquette et prototype interactif

C’est l’étape de web design du projet. L’objectif est de proposer une représentation graphique afin de visualiser à quoi ressemblera le site. Pour le web design j’utilise des outils et des méthodes modernes comme Adobe XD et son système de prototypage interactif qui permet de tester le site comme s’il était déjà en ligne! Grâce à cet outil nous pouvons ajouter des animations, simuler des fonctionnalités et lier les maquettes des différentes pages entre elles. C’est le meilleur moyen pour se projeter.

L'intégration des templates

Il s’agit de transformer les maquettes graphiques en code pour l’afficher dans les navigateurs. Pour intégrer les pages il faut maîtriser les langages HTML, CSS pour l’affichage et JavaScript pour les animations. Lors de l’intégration web il est indispensable de rendre les pages responsives pour que le site s’affiche correctement sur toutes les tailles d’écran.
